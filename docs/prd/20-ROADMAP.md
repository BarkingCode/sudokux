* Offline puzzles (9×9)
* Skia board rendering
* Number pad
* Notes
* Timer

* Push notifications
* Streak tracking

## Phase 4 — Deep UX

* Chapter mode
* Chapter map
* Smart hints
* XP progression

## Phase 5 — Integrations

* Game Center
* Achievements
* Backend sync
* Custom leaderboards

## Phase 6 — Enhancements

* Themes
* Seasonal events
* Advanced animations
